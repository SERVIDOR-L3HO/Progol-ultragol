# Progol-ultragol

## Project Overview
A minimal Node.js web application.

## Tech Stack
- **Runtime:** Node.js 20
- **Server:** Built-in `http` module
- **Entry Point:** `Index.js`

## Project Structure
- `Index.js` — Main server file, serves an HTML page on port 5000
- `README.md` — Project title

## Running the App
The server starts with `node Index.js` and listens on `0.0.0.0:5000`.
